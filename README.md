# myShell

(1) 명령어 실행(Command execution) 

	[prompt] command





(2) 후면 실행(Background execution) 

	[prompt] command &



(3)  종료

	[prompt] exit  또는

	[prompt] logout



(4)  파이프(한개만 처리)

	[prompt] command | command2



(5) Redirection(입출력 모두)

	[prompt] cmd < input > output



추가기능



(6) Background process 관리

	[prompt] myjobs

            현재 background 명령어들과 그들의 process ID를 출력
